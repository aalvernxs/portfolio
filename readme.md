# 💻 Portfólio — Arthur Alvernaz

Portfólio pessoal desenvolvido do zero com HTML, CSS e JavaScript puro,
como projeto de aprendizado para praticar os fundamentos do desenvolvimento web.

## 🛠️ Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## 📌 Seções
- **Hero** — apresentação com cards de informações
- **Skills** — conhecimentos técnicos
- **Experiência** — linha do tempo profissional e acadêmica
- **Cursos** — cursos complementares em andamento
- **Contato** — links para e-mail, LinkedIn e WhatsApp

## ✨ Funcionalidades
- Cursor personalizado com efeito de inércia
- Animações de entrada ao rolar a página
- Navbar com efeito de vidro ao rolar
- Grid animado no fundo da hero
- Layout responsivo
